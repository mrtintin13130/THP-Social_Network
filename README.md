# THE GOSSIP PROJECT

## PRODUCTION

Voici le lien heroku pour voir à quoi cela peut ressembler en production :

[Cliquez-ici](https://thp-social-gossip.herokuapp.com/)

## INSTALLATION

Une fois le dépot cloné:

```
cd [nom-du-repo]

rails db:create

rails db:migrate ```

## UTILISATION

Pour lancer le serveur en local :

```
rails server ```
